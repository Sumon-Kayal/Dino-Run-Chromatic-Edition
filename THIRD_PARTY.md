Usage in This Project

The following components are derived from or inspired by the Chromium Dino game:

- Game logic concepts (physics, jump mechanics, speed scaling)
- Obstacle system (gap calculation, spawn timing)
- Collision detection structure (multi-box AABB approach)
- Audio assets:
  - "jump"
  - "die"
  - "milestone"

These components have been modified, refactored, and integrated into a new modular architecture.
